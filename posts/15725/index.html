解决无法安装的问题

配置 C++ build tools 环境

以管理员权限打开 cmd 或 PowerShell 运行以下命令：

npm install --global windows-build-tools --vs2015
  • 如果你的系统提示找不到 npm 命令，你可能需要安装 Node.js。

设置环境变量

打开 cmd 运行以下命令：

SET PATH=%PATH%;C:\Program Files (x86)\Windows Kits\8.1\bin\x86
SET CL=-FI"C:\Program Files (x86)\Microsoft Visual Studio 14.0\VC\include\stdint.h"

安装 pycrypto

打开 cmd 或 PowerShell 运行以下命令：

pip install pycrypto

解决无法使用的问题

  1. 打开 <your_python_path>\Lib\site-packages\Crypto\Random\OSRNG\nt.py 文件
    • <your_python_path>：python的安装目录，例如 D:\Program Files\Python\Python37
  2. 找到以下内容：
    import winrandom
    替换成：
    from . import winrandom
